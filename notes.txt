how to run this app:
1. npm install
2. npm run start
3. json-server should be installed globally of locally for api data
4. run this command which start json server for us
5. json-server --port 3004 --routes src\apiCalls\fakeData\routes.json src\apiCalls\fakeData\testdata.json
